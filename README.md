# map-tool



| 缩放级别 | 显示效果 | 用途 |
|---------|----------|------|
| 10 – 12 | 城市级 | 大区域概览，适合宏观规划 |
| 14 – 16 | 街道级 | 常规地图查看，能看清主要道路 |
| 17 – 18 | 高精度 | 建筑物轮廓、细部道路详情 |
| 19 | 超高精度 | 极度清晰（注意：部分偏远地区可能无此级别数据） |
